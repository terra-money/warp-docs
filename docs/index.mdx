# Warp Docs

Warp is an open-source, [decentralized event handler](./how/overview.mdx) for Cosmos blockchains, allowing users to place executable [Job messages](./how/overview.mdx#jobs) in a queue to be triggered whenever a specific set of [conditions](./how/overview.mdx#conditions) is met. Using a decentralized system of smart-contract-controlled parameters and reward-incentivized [keepers](./how/overview.mdx#keepers), Warp unlocks the future of blockchain automation.
